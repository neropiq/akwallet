Aidos Wallet (AKWallet)
=====

## Overview

This is the wallet for Aidos Kuneen step2.


## Requirements

* git
* go 1.9+
* yarn
are required to compile.


## Install
    $ git clone github.com/AidosKuneen/akwallet
    $ cd gui
    $ yarn install
    $ yarn build
    $ cd ..
    $ go generate ./...
    $ go build
