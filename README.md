Aidos Wallet (AKWallet)
=====

## Overview

This is the wallet for Aidos Kuneen step2.

UNDER CONSTRUCTION. DON'T TOUCH ME.

## Requirements

* git
* go 1.9+
* yarn

are required to compile.


## Install
    $ git clone github.com/AidosKuneen/akwallet
    $ cd frontend
    $ yarn install
    $ yarn build
    $ cd ..
    $ go generate ./...
    $ go build
